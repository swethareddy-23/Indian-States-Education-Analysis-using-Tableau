# 📊 Indian States Education Analysis using Tableau

## 🔹 Project Overview
This project analyzes education data across different Indian states using Tableau. The objective is to visualize key education metrics and identify patterns, trends, and disparities among states through an interactive dashboard.

## 🔹 Tools Used
- Tableau
- Data Visualization
- Data Cleaning
- Dashboard Design

## 🔹 Dataset
The dataset includes:
- State-wise education statistics
- Literacy rates
- Enrollment data
- Gender-wise education details
- Other key education indicators

## 🔹 What I Did
- Cleaned and prepared the dataset for analysis
- Imported data into Tableau
- Created interactive dashboards and visualizations
- Used filters and parameters for dynamic analysis
- Compared education metrics across states
- Highlighted trends and patterns visually

## 🔹 Key Analysis Performed
- State-wise literacy rate comparison
- Gender-wise education analysis
- Enrollment trends across states
- Identification of top and low-performing states
- Visualization of education disparities

## 🔹 Key Insights
- Identified states with highest and lowest literacy rates
- Observed gender gap in education across regions
- Found trends in enrollment and education growth
- Highlighted areas needing improvement

## 🔹 Dashboard Features
- Interactive filters
- Dynamic charts and maps
- Easy comparison between states
- User-friendly design

## 🔹 Conclusion
This project demonstrates how Tableau can be used to transform raw data into meaningful visual insights for better decision-making in the education sector.
